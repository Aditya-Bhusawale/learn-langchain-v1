# LangChain v1 Learning Repository

This repository contains hands-on examples and projects built using LangChain v1.

## Features

* 🚀 LangChain v1 Architecture
* 🤖 LLM Integration (Groq, OpenAI, Ollama, etc.)
* 📝 Prompt Templates and Prompt Engineering
* 🔧 Tool Creation and Tool Calling
* 🧠 AI Agents with `create_agent()`
* 💬 Chat Models and Message Handling
* 📚 Structured Output with Pydantic Schemas
* 🔄 Streaming Responses
* 🔄 Batch Processing
